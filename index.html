Coherent Rayleigh–Sommerfeld summation over discretized aperture sources, multi-frequency averaged. Homogeneous tissue, c = 1540 m/s, 0.5 dB/cm/MHz attenuation.
Elevation plane. Focus comes from a fixed acoustic lens (rectangular aperture) — a 1-D array has no electronic control here, so this plane is always softer than x–z.
One-way transmit lateral beam profile near the focal depth; FWHM read at −6 dB. The steered trace exposes grating lobes when element pitch exceeds λ/2.
Butterworth–Van Dyke lumped-element model (thickness mode). Grey/red dotted lines mark resonance fr and anti-resonance fa; the R₁/Qm ratio sets bandwidth. The dashed amber line is the 50 Ω system reference — distance from it shows how much electrical matching a drive channel needs. Only C₀ (∝ electrode area) changes with geometry; fr, fa and R₁ are set by thickness and material.
Fixed reflectivity map (wire points + anechoic cyst + speckle) convolved with a depth-dependent lateral PSF of width w₀ = 0.886·λ·F#. Sharpness is set by f0, aperture, focus and apodisation (via λ and F#). If the array is undersampled (pitch > λ/2) each target also casts mirrored grating-lobe ghosts at ±z·tan(θg), suppressed to the element-factor level (~−19 dB); lowering element count over a fixed aperture makes them visible. Material and elevation do not enter this plane. A resolvability demo, not a scattering simulation.
